# To-Do List Scheduler

A simple command-line to-do list app built with Python.

## Features
- Add, edit, and delete tasks
- Urgency levels (optional, nice to have, must do, urgent)
- Deadline tracking with overdue detection
- Tasks are saved locally in JSON format

## Requirements
- Python 3.x

## How to Run
python todolist-scheduler.py

## Usage
- N — Add new task
- C — Change existing task
- D — Delete task
- Q — Save and quit
